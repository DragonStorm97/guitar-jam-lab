# Guitar Jam Lab

Interactive static guitar study page for scales, tunings, fretboard maps, CAGED shapes, harmony ideas, generated tab charts, and blues practice.

## GitHub Pages

This repo is intended to publish directly with GitHub Pages. The app entry point is:

```text
index.html
```

If Pages is not enabled yet, use GitHub repo settings:

1. Open **Settings**.
2. Open **Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Select branch **main** and folder **/**.
